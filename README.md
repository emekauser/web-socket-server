# web-socket-server
A small websockets app for sharing arbitrary messages with all other users with mongodb as database

# How to user

1. Clone the repo and cd into the folder
2. ``` npm install ```
3. ``` npm start  ```

# End points
1. get_all_messages GET
2. get_active_users GET

# Socket Event
1. start
2. error
3. message
4. disconnect
